Animation Guidelines
css/* Smooth, confidence-inspiring transitions */
@keyframes pulse {
  0%, 100% { boxShadow: '0 0 20px rgba(0, 212, 255, 0.2)'; }
  50% { boxShadow: '0 0 40px rgba(0, 212, 255, 0.4)'; }
}

/* All transitions should be smooth like aircraft movements */
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
Visual Effects

Subtle particle field - Like stars visible from cockpit at altitude
Soft glows on interactive elements - Like backlit cockpit buttons
No harsh edges - Everything has subtle borders with glow
Glass morphism on cards - Like looking through aircraft windows

The Overall Feel
The interface should feel like:

A modern aircraft instrument panel
Trustworthy and technical but not intimidating
Calming through its precision
Professional aviation software meets meditation app
The bridge between human anxiety and cosmic understanding

Every element should whisper "I know exactly what I'm doing" while being approachable enough for someone having a panic attack to navigate.
